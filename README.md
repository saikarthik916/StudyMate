# 📚 StudyMate

> _The repository that contains the demo of the project "Student assistant"._

---

## 🚀 Overview

**StudyMate** is a web-based student assistant platform designed to help students organize, access, and manage academic materials by semester, branch, regulation, and subject. It allows you to **login/signup**, browse and upload files, and fetch relevant study resources conveniently through an interactive interface.

---

## 🌟 Features

- 👩‍🎓 **Student Registration & Login:**  
  Secure sign-up and login functionality backed by MongoDB and Express.js.

- 🗂️ **Semester & Subject Browser:**  
  Browse subjects and resources filtered by semester, branch, and regulation.

- ☁️ **File Upload & Management:**  
  Upload, manage, and view files related to subjects (assignments, notes, etc).

- 💾 **Teacher/Admin Role:**  
  Teachers can upload and manage materials.  
  _To experience the teacher role, log in as:_  
  **Username:** `vijay`  
  **Password:** `12345`

- 🛡️ **Modern Backend Stack:**  
  Powered by Node.js, Express.js, and MongoDB for robust API and data management.

- 🔍 **Responsive UI:**  
  Stylish, user-friendly interface built with HTML, CSS, and JavaScript.

## 🛠️ Setup & Usage

1. **Clone this repository:**
   ```bash
   git clone https://github.com/VHITMAN2025/StudyMate.git
   cd StudyMate
   ```

2. **Install Backend Dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**  
   Create a `.env` file for database credentials as per your setup, or adjust connection strings in the JS files.

4. **Start the Server:**
   ```bash
   node server.js
   ```

5. **Open your browser at:**  
   ``````
   http://localhost:PORT
   ```
   (Replace `PORT` with your configured port, e.g., 3000 or 1431.)
